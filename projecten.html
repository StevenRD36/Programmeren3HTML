In het eerste jaar werden er een paar projecten gemaakt. Deze waren dan tegelijkertijd ook het examen waarop men werd beoordeeld.

Databanken

Voor het vak databanken, was het de bedoeling om zelf een databank aan te maken. Dit kon zowel gebeuren via de SQL-commando's als via de interface. Deze opdracht was ook tegelijkertijd, het examen voor databanken. Deze opdracht bestond uit 3 onderdelen:

  1. DDL(Data Definition Language)

    2. DDL kon zowel via SQL-commando's gedaan worden als via de interface. Voor dit onderdeel heb ik gewerkt via de interface.

  2. DML(Data Manipulation Language)

    3. Voor DML heb ik gebruik gemaakt van de sQL-commando's. Dit was het gemakkelijkste om mee te werken, want er diende data veranderd te worden. Om op deze manier te werken, was dit het eenvoudigste. Zo was je er zeker van dat de nodige data, op de correcte manier gewijzigd was.

Programmeren 1

Voor programmeren 1, was het de bedoeling om een project te maken, hierbij kwamen alle aangeleerde vaardigheden van het programmeren aan bod. Dit project was tegelijkertijd ook het examen voor deze cursus. De opdracht bij dit project was om een keuzemenu te maken waarbij de gebruiker kon kiezen tussen verschillende opties. Er werd ook gebruik gemaakt van een databank omdat er data uit de databank werd gehaald en werd weergegeven in de output.
